# ScreenMarker Pro 快速开始指南

欢迎使用 **ScreenMarker Pro**！

本文档将帮助您快速了解如何安装、运行和使用这款极简的 macOS 屏幕标注工具。

## 打开并运行项目

### 1. 用Xcode打开项目

```bash
cd "/Users/mpp/Documents/AI Coding/ScreenMarkPro"
open ScreenMarkPro.xcodeproj
```

### 2. 配置开发团队（首次运行必需）

在Xcode中：

1. 选中左侧项目导航器中的**ScreenMarkerPro**项目图标
2. 选择**Targets** → **ScreenMarkerPro**
3. 点击**Signing & Capabilities**标签
4. 在**Team**下拉菜单中选择您的Apple开发者账号（或选择个人团队）

### 3. 运行应用

按`Cmd + R`或点击顶部的播放按钮。

### 4. 预期效果

应用启动后，您会看到：

- ✅ 应用以后台模式运行（无Dock图标）
- ✅ 屏幕中心出现一个**400×300**的矩形边框
- ✅ 边框呈现**激光蓝→霓虹紫**的渐变效果
- ✅ 边框在所有应用之上显示（包括全屏应用）

```
ScreenMarkerPro/

```

## 技术亮点

### 1. OverlayWindow配置

```swift
self.level = .screenSaver              // 最高优先级
self.ignoresMouseEvents = true         // 点击穿透
self.collectionBehavior = [
    .canJoinAllSpaces,                 // 跨桌面显示
    .fullScreenAuxiliary               // 全屏辅助窗口
]
```

### 2. 渐变边框实现原理

```
CAGradientLayer (渐变色)
    └── mask: CAShapeLayer (矩形框路径)
        → 最终效果：渐变色的矩形边框
```

## 常见问题

### Q1: 运行后看不到矩形框？

**A**: 检查Xcode控制台是否有错误。确保应用成功启动且没有权限弹窗。

### Q2: 如何退出应用？

**A**: 在Xcode中点击停止按钮，或使用`Cmd + Q`（如果应用获取了焦点）。

### Q3: 为什么要设置LSUIElement？

**A**: 使应用以后台方式运行，不在Dock中显示图标，符合辅助工具的使用习惯。
